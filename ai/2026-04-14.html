5. Стэнфордский AI Index 2026: Китай почти догнал США по качеству моделей

1. Claude Code v2.1.105 — масштабный релиз с десятками исправлений
Вышел 13 апреля 2026 года. Добавлены: параметр `path` для инструмента EnterWorktree, хук PreCompact (может блокировать сжатие контекста), фоновые мониторы для плагинов через ключ `monitors` в манифесте, алиас `/proactive` для `/loop`. Исправлены десятки багов: зависание стримов API, неправильная обработка rate-limit ошибок, падения при resume, проблемы с MCP-инструментами и плагинами, баги рендеринга терминала.
Claude Code releases
2. Claude Code Week 15 — Ultraplan, инструмент Monitor, /autofix-pr
Дайджест за 6–10 апреля 2026 (версии v2.1.92–v2.1.101). **Ultraplan** в research preview: запускаешь планирование из терминала, Claude строит план в облаке, ты комментируешь его в браузере, затем запускаешь удалённо или возвращаешь в CLI. **Monitor** — новый встроенный инструмент: запускает фоновый наблюдатель и стримит события прямо в разговор. **`/loop`** теперь сам выбирает интервал, если его не указать. **`/autofix-pr`** — авто-исправление PR прямо из терминала: Claude следит за CI и комментариями ревью и пушит исправления до зелёного статуса. **`/team-onboarding`** генерирует гайд по онбордингу на основе твоего реального использования Claude Code.
