OpenAI, ChatGPT и Codex

1. ChatGPT начал раскатывать рекламу на Free и Go в Австралии, Новой Зеландии и Канаде
В свежих ChatGPT release notes OpenAI зафиксировала первый заметный рекламный rollout для потребительских тарифов Free и Go в трёх англоязычных рынках. Параллельно компания отдельно подчёркивает, что Plus, Pro, Business, Enterprise и Education остаются без рекламы.

2. OpenAI ввела новый Pro за $100 и переразметила лимиты Codex
В тех же release notes OpenAI описывает новый тариф Pro за $100 в месяц для длинных и интенсивных Codex-сессий. Временный бонус даёт до 10x больше Codex-использования, а лимиты Plus при этом перераспределяют в пользу более равномерной недельной нагрузки вместо редких длинных сессий.
ChatGPT Release Notes
3. GPT-5.4 mini стал fallback-моделью, а GPT-5.1 окончательно убрали из ChatGPT
В model release notes OpenAI закрепила GPT-5.4 mini как fallback для GPT-5.4 Thinking и одновременно завершила вывод GPT-5.1 Instant, Thinking и Pro из ChatGPT. Это важный сдвиг для реального пользовательского маршрута: даже без ручного выбора модельная линейка заметно упростилась.
Model Release Notes
4. OpenAI закрыла раунд на $122 млрд при оценке $852 млрд
В официальном посте от 31 марта OpenAI объявила о $122 млрд committed capital и пост-манй оценке в $852 млрд. Компания прямо описывает ChatGPT, API и Codex как взаимно усиливающуюся инфраструктурную систему, где compute становится главным стратегическим плечом роста.
OpenAI
5. Enterprise уже даёт OpenAI более 40% выручки, у Codex - 3 млн weekly active users
В заметке CRO Denise Dresser OpenAI пишет, что enterprise-направление уже формирует более 40% выручки и к концу 2026 года может выйти на паритет с consumer. Там же зафиксированы операционные масштабы: Codex достиг 3 млн еженедельных активных пользователей, а API проходит более 15 млрд токенов в минуту.
OpenAI
6. Вышел Codex CLI 0.122.0 со свежим stable-релизом и большой порцией TUI/Sandbox изменений
В stable-релизе `0.122.0` появились `/side`-диалоги, запуск реализации из fresh context в Plan Mode, заметное расширение plugin workflow и более жёсткие sandbox/permissions-политики. По объёму changelog это один из самых плотных выпусков Codex CLI за последние дни, а уже 21 апреля поверх него пошли новые alpha-сборки `0.123.0-alpha.*`.
