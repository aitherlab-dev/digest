1. NVIDIA выпустила Ising — первые открытые квантовые AI-модели
NVIDIA запустила семейство моделей Ising — первые в мире открытые AI-модели для квантовых вычислений. Ising Calibration (35 млрд параметров) автоматизирует калибровку квантовых процессоров, сокращая дни работы до часов. Ising Decoding в 2,5 раза быстрее и в 3 раза точнее существующих инструментов коррекции квантовых ошибок.
NVIDIA
2. Claude Code получил масштабный редизайн и функцию Routines
Anthropic переработала десктопное приложение Claude Code: встроенный терминал, редактор файлов, просмотрщик диффов, поддержка параллельных сессий. Главная новинка — Routines: автоматические задачи по расписанию, через API или триггером с GitHub. Работают на облачной инфраструктуре.

3. Meta представила Muse Spark — первую закрытую фронтирную модель
Meta Superintelligence Labs запустила Muse Spark — первую проприетарную фронтирную модель компании без открытых весов. Поддерживает текст, изображения и речь нативно, использует «компрессию мыслей» для высокого качества при десятикратно меньших вычислениях. В рейтинге Intelligence Index — 4-е место.
