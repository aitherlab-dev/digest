AI и генеративные медиа

14 мая 2026

Видео, изображения и релизы AI-лабораторий

1. Runway представила Runway Agent для законченных видео из диалога
Официальный аккаунт Runway описал Runway Agent как творческого партнёра, который помогает придумать ролик и довести его до готового видео со звуком и монтажом через разговорный интерфейс. Это важный сдвиг для видеогенерации: продукт продаётся уже не как одиночная модель клипа, а как полный рабочий процесс для рекламы, коротких роликов и социальных форматов.
X / Runway
2. Higgsfield показал Supercomputer как агентный слой для image/video моделей
Higgsfield пишет, что Supercomputer работает из браузера или Telegram, принимает задачу текстом, оркестрирует LLM и модели изображений/видео, анализирует видео и аудио и выполняет цепочку через 40+ инструментов. Практический смысл в том, что генерация медиа становится не кнопкой «сделай картинку», а управляемой фабрикой задач с памятью и инструментами.
X / Higgsfield
3. Anthropic подняла недельные лимиты Claude Code на 50%
Официальный аккаунт ClaudeDevs сообщил, что лимиты Claude Code увеличены на 50% до 13 июля для Pro, Max, Team и Enterprise с оплатой по местам. Это продуктовый релиз для интенсивной агентной разработки: больше длинных сессий помещается в подписочный лимит без немедленного перехода к отдельной оплате.
X / ClaudeDevs
4. Anthropic отделяет Agent SDK и `claude -p` в отдельный месячный кредит
С 15 июня платные планы Claude смогут получать отдельный месячный кредит для программного использования: Claude Agent SDK, `claude -p`, Claude Code GitHub Actions и сторонних приложений на SDK. Для локальных harness и автоматических задач это важная смена экономики: интерактивный Claude Code и программные запуски начинают жить в разных лимитных корзинах.
X / ClaudeDevs
5. Claude Opus 4.7 получил быстрый режим в Claude Code и API preview
ClaudeDevs подтвердили предварительный быстрый режим для Claude Opus 4.7 в API и Claude Code, а также отдельный waitlist для API-пользователей. Это не новая базовая модель, но важное изменение доступности: Anthropic пытается сделать флагманский Claude полезнее в агентных сценариях, где задержка и стоимость каждого длинного прохода быстро становятся ограничением.
