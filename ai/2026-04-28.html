OpenAI, ChatGPT и Anthropic

1. OpenAI и Microsoft переписали партнёрство: меньше эксклюзивности, больше cloud-flexibility
OpenAI официально объявила amended agreement с Microsoft. Microsoft остаётся primary cloud partner, но OpenAI теперь может предоставлять продукты клиентам через любого cloud provider; лицензия Microsoft на IP OpenAI становится non-exclusive до 2032 года, а revenue share от OpenAI к Microsoft продолжается до 2030 года с cap. Для рынка это важнее обычного партнёрского пресс-релиза: OpenAI получает больше свободы для AWS, Google Cloud и enterprise-сделок, а Microsoft фиксирует долгосрочный доступ и долю участия.

AI и Tech радар

1. GitHub переводит Copilot на usage-based billing с 1 июня
GitHub объявил, что все планы Copilot переходят на usage-based billing 1 июня 2026 года. Вместо старой логики premium requests появятся GitHub AI Credits, расход которых считается по token usage, включая input, output и cached tokens; code review также начнёт затрагивать GitHub Actions minutes. Это ключевой поворот монетизации coding assistants: агентные и тяжёлые модели перестают помещаться в простую подписку с мягкими лимитами.
GitHub Blog · GitHub Docs

Это важный enterprise-сигнал рядом с OpenAI/Microsoft сделкой: рынок уже уходит от пилотов к массовому rollout, где цена, обучение сотрудников, governance и измеримый эффект становятся главнее демо.

4. Google DeepMind описала Decoupled DiLoCo для устойчивого distributed training
Google DeepMind опубликовала Decoupled DiLoCo — подход к resilient, distributed AI training at scale. Идея в том, чтобы группы workers могли идти в разном темпе, переживать stragglers, partial failures и слабые network links без остановки всей тренировки. Это инфраструктурная тема, но именно такие вещи определяют, насколько реально обучать большие модели на географически распределённом и неоднородном железе.
Google DeepMind · HN
5. Microsoft VibeVoice ASR снова всплыл как practical open speech model
Simon Willison отметил MIT-licensed VibeVoice ASR как сильную альтернативу Whisper-сценариям со speaker diarization. Microsoft описывает модель как unified speech-to-text для до 60 минут аудио за один проход с transcription, speaker diarization и timestamping в едином inference path. Практический смысл: длинные встречи, подкасты и интервью можно обрабатывать структурно, без отдельной цепочки ASR + diarization + склейка.
Microsoft Community Hub · Simon Willison

8. Utilyze спорит с nvidia-smi и nvtop о том, что значит “GPU utilization”
Systalyze выпустила open-source GPU monitoring tool, который измеряет compute и memory throughput через hardware performance counters, а не только факт активности kernel на GPU. Автор утверждает, что стандартные метрики вроде nvidia-smi могут показывать 100% utilization при реальном useful compute throughput в районе 1-10%. Для AI-инфраструктуры это важная тема capacity planning: неправильная метрика ведёт к неправильной покупке железа.
