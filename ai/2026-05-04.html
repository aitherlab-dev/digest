AI и Tech радар

1. Agentic Coding Is a Trap стал главным HN-тредом про пределы coding agents
HN активно обсуждает эссе Lars Faye о том, что agentic coding может создавать иллюзию ускорения, но переносит сложность в review, context drift и responsibility. Это важный counter-signal на фоне ежедневного agent hype: качество workflow всё ещё зависит от проверки, boundaries и умения остановить неверную ветку работы.
HN · Оригинал
2. DeepClaude обещает Claude Code agent loop на DeepSeek V4 Pro дешевле в 17 раз
Проект `aattaran/deepclaude` вышел в топ HN как попытка совместить Claude Code-style agent loop с более дешёвым backend на DeepSeek V4 Pro. Даже если практическое качество надо проверять руками, сам интерес показывает главный pressure point рынка coding agents: стоимость long-running sessions быстро становится архитектурным ограничением.
HN · GitHub
3. HN обсуждает исследование, где OpenAI o1 лучше врачей в ER triage diagnoses
Guardian-материал попал в HN с цифрой: o1 правильно диагностировал 67% ER patients против 50-55% у triage doctors в Harvard trial. Это high-stakes AI story: результаты интересны, но требуют осторожного чтения, потому что clinical workflow зависит не только от diagnosis accuracy, но и от liability, data quality и integration в больничную систему.
HN · The Guardian
4. Show HN: Apple SHARP запускают в браузере через ONNX Runtime Web и WebGPU
Автор demo экспортировал Apple SHARP, single-image 3D Gaussian splatting model, в ONNX и запустил inference локально в браузере. Важная часть — image не покидает tab, а результат можно скачать как `.ply`; caveat в том, что model sidecar около 2.4 GB и веса Apple имеют research-use license.
HN · GitHub
5. BYOMesh показывает LoRa mesh radio с заявкой на 100x bandwidth
HN вывел в топ BYOMesh: новый LoRa mesh radio проект, который обещает существенно более высокую bandwidth, чем привычные low-bandwidth mesh setups. Для off-grid/community networking это интересный сигнал, потому что mesh-сети часто упираются не в routing, а в usable throughput для реальных приложений.
HN · Оригинал
6. Citizen Lab описал global telecom exploitation by covert surveillance actors
Тред Bad Connection на HN указывает на исследование Citizen Lab о covert surveillance actors, которые эксплуатировали telecom infrastructure для tracking. Это security story не про один app bug, а про уровень телеком-сетей, где visibility и accountability для пользователей почти отсутствуют.
HN · Citizen Lab
7. “LLMs Are Not a Higher Level of Abstraction” снова формулирует скепсис к codegen
HN обсуждает текст, где LLM предлагается считать не новым abstraction layer, а вероятностным interface с собственными failure modes. Это полезный framing для инженеров: если abstraction не даёт стабильных guarantees и composability, её надо использовать как помощника, а не как фундаментальную замену design.
